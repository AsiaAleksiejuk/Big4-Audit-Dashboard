# 📊 Big 4 Audit Dashboard — Excel & Power BI Project

## 📌 Project Overview
Interactive analysis of audit performance across Big 4 firms:
- Deloitte
- PwC
- KPMG
- Ernst & Young

The project explores fraud detection, revenue impact, audit effectiveness, and AI adoption trends.

---

## 🛠 Tools Used
- **Excel**
  - Data cleaning
  - Pivot tables
  - Aggregations
- **Power BI**
  - Interactive dashboard
  - DAX measures
  - KPI cards
  - Trend analysis
  - Slicers and filters

---

## 📊 What the Project Shows

### 🔍 Key Metrics
- Total audit engagements: **55,000+**
- Fraud cases detected: **5,270**
- Revenue at risk: **$27B+**
- Average effectiveness score: **7.49**

---

### 📈 Analysis Includes
- Fraud trends (2020–2025)
- Revenue impact by firm
- Effectiveness vs AI usage
- Industry breakdown:
  - Tech
  - Healthcare
  - Finance
  - Retail
- AI adoption over time
- Workload vs satisfaction relationship

---

### 🤖 Key Insight
Use of AI in auditing slightly improves effectiveness and satisfaction, while reducing workload.
